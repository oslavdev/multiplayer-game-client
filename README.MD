# Multiplayer shooter game

WebGL sci-fi multiplayer shooter game.

## Features

- Select and customize character
- Private maps
- Choose weapons
- Audio, video and written chat
- Selection of maps

## Tech

- Turborepo
- Next.js
- Node.js 
- Three.js / React-three-fiber
- GLSL
- Docker
- GraphQL
- PostgreSQL
- Ably (realtime communication)
- Prisma
